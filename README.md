# eMarket - E-Commerce Platform

## Admin Login Details

To access the admin panel, use the following credentials:

- **URL**: http://localhost/AI/views/admin/login.php
- **Email**: admin@emarket.com
- **Password**: password

## Features

- Product Management
- Category Management  
- Order Management
- User Management
- Banner Management
- M-Pesa Payment Integration

## Installation

1. Run the install.php file to set up the database
2. Access the admin panel at /views/admin/login.php
3. Login with the credentials above

## Tech Stack

- PHP
